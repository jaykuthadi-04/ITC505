Addendum

Challenges Faced and Solutions:

  • Ensuring Random but Solvable Board: Randomly toggling tiles using the same logic as user clicks (simulate clicks) guaranteed solvable configurations.
  • CSS Styling for Visual Clarity: Created a clear visual distinction between on and off states using contrasting colors. Ensured responsiveness for different screen sizes.
  • Simulating clicks was key to maintaining solvability.
  • Modular code allowed easier debugging and feature addition, such as the restart button.
